Перед початком роботи над розробкою необхідно:
1.Встановити .NET SDK 8
2.Стягнути поточну версію main гілки
3.Заповнити особисту інформацію в файл, непомітний для git:
 -скопіювати вміст файлу appsettings.example.json в appsettings.Development.json
 -замінити заповнювачі на реальні дані 
4.(опційно) Встановити розширення для VS Code: IntelliCode for C# Dev Kit
5.Запустити команду для встановлення необхідних інструментів: dotnet tool restore
6.Готово

Для запуску додатку слід використовувати команду: dotnet run

Перед виконанням коміту слід відформатувати код: dotnet format

Для загальної конфігурації використовуйте appsettings.json, для особистих даних - appsettings.Development.json(appsettings.example.json).

Якщо ви хочете додати новий інструмент, використовуйте команду для коректного його додавання в залежності проекту: dotnet tool install toolname
